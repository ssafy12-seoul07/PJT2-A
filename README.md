### 영상 페이지

- [x] 영상 페이지에 들어가면 비디오 id에 해당하는 db 생성
- [x] 조회할 때마다 조회 수 증가
- [x] 비디오에 대한 댓글 db 추가
- [ ] (심화) 비디오 찜기능

**[영상 플레이어]**

- [x] iframe 안에 id에 해당하는 유튜브 영상 삽입

**[영상 댓글]**

- [x] 영상에 대한 댓글의 생성기능. 유저id, 내용, 타임스탬프 기록. 최신순 정렬.
- [x] 영상에 대한 댓글은 모두 읽기 가능
- [ ] 로그인한 유저가 댓글 작성자라면 수정 및 삭제 가능. 수정 및 삭제 버튼 노출.
- [ ] (심화) 유저정보에 내가 쓴 댓글 추가
